Calculate Hash

Each request sent to offer-ready requires a "hash". A hash is a text string created by a cryptographic function, used to authenticate the integrity of the request. A distinct hash is calculated using the environment (Preview or Live), the name of the endpoint and the "Secret Key" as an input. Optionally (depending on individual configuration) the calculation will include further input parameters sent along with the request.

Usually the hash is implemented as a hidden input field in the web form submitting the request. Note that the hash changes whenever you create a new "Secret Key"!
