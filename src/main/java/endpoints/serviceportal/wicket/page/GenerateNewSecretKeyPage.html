Create a New Secret Key

Each Offer-Ready account has a randomly created "Secret Key", which is used to calculate cryptographic hash strings to access the different endpoints of this configuration (see "Hash Calculator").

  1. Click the button to create a new secret key.
  2. The new secret key will not be in effect immediately. It will be available in a new revision of the configuration. After creating the new secret key, you will have to publish that new revision.
  3. Use the "Hash Calculator" to calculate the new hash strings and adopt your web forms accordingly. Note that the hash is different for PREVIEW and LIVE environments!
